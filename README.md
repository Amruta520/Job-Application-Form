A simple and responsive Contact / Job Application Form built using HTML, CSS, and JavaScript.
The form includes client-side validation to ensure users enter correct information before submitting.

--Features--

Responsive form design

Modern gradient background UI

Input validation using JavaScript

Email format validation

Phone number validation (10 digits)

Error messages for incorrect inputs

Success message after submission

Form reset after successful submit

--Technologies Used--

HTML5 – Structure of the form

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Form validation and interactivity
